# Trademarks

**"Celiums" and "Celiums Memory"**, and the Celiums logo, are
trademarks of **Celiums Solutions LLC**.

The Apache-2.0 license that governs the source code in this repository
does **not** grant rights to use the trademarks. This is standard
open-source hygiene: the code is open; the name and logo are not.

## What you can do without asking

- Say your product integrates or is compatible with Celiums Memory,
  where that is true (*"Powered by Celiums Memory"*, *"Compatible with
  Celiums Memory"*).
- Cite Celiums in docs, articles, papers, talks, reviews, tutorials,
  and comparisons. Normal attribution conventions apply.
- Use the name descriptively and factually
  (*"Celiums Memory is an open-source memory engine"*).
- Acknowledge Celiums as upstream in a fork's README/NOTICE.

## Forks

Apache-2.0 lets you fork the code. It does not let the fork use the
marks. If you fork: **rename it** (don't imply it is official Celiums —
e.g. not *"Open Celiums"*, *"Celiums Community Edition"*), remove the
Celiums logo from your README/UI, and keep the upstream NOTICE per
Apache-2.0 §4. Trademark removal and attribution are compatible
obligations. A single-line *"originally based on Celiums Memory"* is
fine and needs no permission.

## What needs permission

Anything that implies Celiums Solutions LLC endorses, sponsors, or
produced your work when it does not; product names built around
"Celiums" that could confuse users about what is official; or use of
the marks in ways that mislead or disparage.
